# 🚗 CAN-SDA  
### CAN 데이터를 활용한 자동차 급발진 탐지 연구  
*(Sudden Acceleration Detection using CAN Data)*

## 📌 프로젝트 개요

자동차 급발진(Sudden Unintended Acceleration) 사고는 명확한 원인이 규명되지 않았음에도 지속적으로 발생하고 있다. 기존의 사고기록장치(EDR, Event Data Recorder)는 사고 전 약 5초, 사고 후 0.3초의 제한된 데이터만을 기록하여 급발진 분석에 한계가 있다.

본 프로젝트는 **테슬라 모델 3의 CAN(Controller Area Network) 데이터**를 활용하여  
**딥러닝 기반 실시간 급발진 탐지 가능성**을 검증하는 것을 목표로 한다.  
RNN 및 LSTM 기반의 지도학습 모델을 적용하여 급발진을 탐지하고, 실시간 모니터링 시스템으로의 활용 가능성을 평가하였다. <br> <br>
본 논문은 한국정보보호학회의 CISC-W'24 에 제 1저자로  "CAN 데이터를 활용한 자동차 급발진 탐지 연구" 주제로 기재되었음.

---

## 🧠 연구 배경

- 급발진 추정 사고 신고 건수: 최근 10년간 462건 (2024년 6월 기준)
- EDR 데이터의 시간적 한계로 인해 급발진 전조 분석이 어려움
- CAN 데이터 기반의 연속적인 시계열 분석 필요성
- 딥러닝 기반 실시간 급발진 탐지 시스템 제안

---

## ⚙️ 연구 프레임워크

1. 테슬라 모델 3 CAN 데이터 수집  
2. 급발진 정의  
   - 대법원 판례 기준 적용  
   - 엔진 RPM이 4,000 이상으로 유지되는 상태를 급발진으로 정의  
3. 데이터 전처리 및 Feature 선정  
   - Timestamp  
   - 차량 속도 (`DI_uiSpeed`)  
   - Torque Sensor 값  
4. 모델 학습  
   - RNN  
   - LSTM  
5. 성능 평가  
   - Accuracy  
   - F1-Score  
   - 탐지 시간  

---

## 🧪 실험 및 결과

- PCA 분석 결과, Timestamp의 연속성이 사라져 예측 성능에 한계가 발생함을 확인
- 시간적 연속성을 반영하기 위해 Torque Sensor 값을 Feature로 추가
- 급가속(급발진)과 일반 주행 간 명확한 패턴 차이를 확인

### 모델 성능 비교

| 모델 | Accuracy | F1-Score | 탐지 시간 |
|------|----------|----------|-----------|
| RNN  | 약 98%   | 약 0.98  | 실시간 가능 |
| LSTM | **99.37%** | **0.9882** | **0.48초** |

LSTM 모델은 RNN 대비 Accuracy 1.11%, F1-Score 1.07% 향상된 성능을 보였다.
